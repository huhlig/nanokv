> NanoKV is a lightweight embeddable single file key value table service.

## Project Structure

```
nanokv/
├── docs/            # Documentation, Architecture, ADRs
├── api/             # Embeddable API
├── cli/             # CLI Utilities
├── net/             # Network Service
├── table/           # Key Value Tables
├── pager/           # File Pager
└── vfs/             # Virtual File System
```
